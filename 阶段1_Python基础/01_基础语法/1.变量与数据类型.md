# 变量与数据类型

## 一、什么是变量？

变量就是一个**名字**，用来指向内存中的某个数据。

```python
name = "张三"     # name 这个名字指向了字符串 "张三"
age = 20          # age 这个名字指向了整数 20
```

**核心理解**：Python 中变量不是"盒子"（存数据），而是"标签"（贴在数据上）。

```python
a = [1, 2, 3]
b = a           # b 和 a 指向同一个列表，不是复制！
b.append(4)
print(a)        # [1, 2, 3, 4]  ← a 也变了，因为它们指向同一个对象
```

### 变量命名规则

| 规则 | 示例 | 说明 |
|------|------|------|
| 字母/下划线开头 | `name`, `_count` | ✅ 合法 |
| 不能数字开头 | `2name` | ❌ 报错 |
| 只能包含字母、数字、下划线 | `user_name`, `age2` | ✅ 合法 |
| 区分大小写 | `Name ≠ name` | 是两个不同变量 |
| 不能用关键字 | `if`, `for`, `class` | ❌ 这些是 Python 保留字 |

### 命名风格约定（PEP 8）

```python
# 变量和函数：小写 + 下划线
user_name = "张三"
def get_user_info():
    pass

# 常量：全大写 + 下划线
MAX_RETRY = 3
DATABASE_URL = "localhost:3306"

# 类名：大驼峰
class UserService:
    pass
```

## 二、四种基本数据类型

### 1. 整数 int

```python
age = 20
count = -5
big_number = 1_000_000    # 下划线分隔，提高可读性，值仍是 1000000

# 常用操作
print(10 / 3)    # 3.3333  普通除法，结果是 float
print(10 // 3)   # 3       整除（地板除），结果是 int
print(10 % 3)    # 1       取余
print(2 ** 10)   # 1024    幂运算
```

**Python 特点**：整数没有大小限制（不像 Java/C 有 int 溢出问题）。

### 2. 浮点数 float

```python
price = 19.99
pi = 3.14159

# ⚠️ 经典坑：浮点数精度问题
print(0.1 + 0.2)           # 0.30000000000000004  ← 不是 0.3！
print(0.1 + 0.2 == 0.3)    # False

# 解决方案：使用 decimal 模块
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2'))    # 0.3
```

**实际场景**：涉及钱的计算，绝不用 float，用 Decimal 或整数（单位：分）。

### 3. 字符串 str

```python
name = "张三"
greeting = '你好'
multi_line = """
这是
多行
字符串
"""

# 常用操作
s = "Hello, World"
print(len(s))           # 12       长度
print(s.upper())        # HELLO, WORLD
print(s.lower())        # hello, world
print(s.split(", "))    # ['Hello', 'World']   按分隔符拆分
print(s.replace("World", "Python"))   # Hello, Python
print(s.strip())        # 去除首尾空白
print(s.startswith("Hello"))  # True
print(s.find("World"))  # 7       返回索引，找不到返回 -1

# 字符串是不可变的！
s[0] = "h"    # ❌ TypeError: 不能修改
```

### 4. 布尔值 bool

```python
is_active = True
is_deleted = False

# 布尔值本质上是整数: True = 1, False = 0
print(True + True)    # 2
print(True * 10)      # 10

# 以下值在 if 中被视为 False（"假值"）：
# False, 0, 0.0, "", [], {}, (), set(), None
# 其他所有值都是 True

if "":
    print("不会执行")
if "hello":
    print("会执行")     # ← 非空字符串为 True
```

## 三、类型查看与转换

### 查看类型

```python
print(type(42))         # <class 'int'>
print(type(3.14))       # <class 'float'>
print(type("hello"))    # <class 'str'>
print(type(True))       # <class 'bool'>

# isinstance() 推荐用于类型判断（支持继承）
print(isinstance(42, int))        # True
print(isinstance(True, int))      # True  ← 因为 bool 是 int 的子类
```

### 类型转换

```python
# str → int
age = int("20")         # 20
# int("abc")            # ❌ ValueError

# int → str
s = str(100)            # "100"

# str → float
price = float("19.99")  # 19.99

# float → int（截断，不是四舍五入）
n = int(3.9)            # 3

# 四舍五入
n = round(3.5)          # 4
```

## 四、动态类型与强类型

Python 是**动态类型 + 强类型**语言：

```python
# 动态类型：变量可以随时改变指向的类型
x = 10        # x 现在是 int
x = "hello"   # x 现在是 str（合法，不报错）

# 强类型：不同类型之间不会自动转换
print("age: " + 20)     # ❌ TypeError: 不能 str + int
print("age: " + str(20)) # ✅ 手动转换
```

| 概念 | 含义 | Python |
|------|------|--------|
| 动态类型 | 变量不需要声明类型 | ✅ 是 |
| 强类型 | 不同类型不能隐式转换 | ✅ 是 |

## 五、None 类型

```python
result = None    # 表示"没有值"

# 判断是否为 None，用 is 而不是 ==
if result is None:
    print("还没有结果")

# 常见用法：函数默认返回 None
def greet(name):
    print(f"你好, {name}")

x = greet("张三")   # 打印"你好, 张三"
print(x)             # None  ← 函数没有 return，返回 None
```

## 六、常见坑总结

| 坑 | 说明 | 解决 |
|----|------|------|
| 浮点数精度 | `0.1 + 0.2 ≠ 0.3` | 用 `Decimal` 或整数运算 |
| 变量是引用 | `b = a` 不是复制 | 用 `b = a.copy()` 或 `b = a[:]` |
| 字符串不可变 | `s[0] = 'x'` 报错 | 用 `s = 'x' + s[1:]` |
| 布尔是整数 | `True + 1 = 2` | 了解即可，避免混用 |
| 类型转换失败 | `int("abc")` 报错 | 用 `try/except` 捕获 |
