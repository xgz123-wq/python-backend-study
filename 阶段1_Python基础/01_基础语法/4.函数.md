# 函数定义与参数

## 一、什么是函数？

函数 = 一段可以**重复使用**的代码块，给它一个名字，需要时调用。

```python
# 没有函数：重复代码
print("=" * 30)
print("欢迎来到系统")
print("=" * 30)
# ... 100 个地方都要这样写

# 有函数：定义一次，到处调用
def welcome():
    print("=" * 30)
    print("欢迎来到系统")
    print("=" * 30)

welcome()   # 调用
welcome()   # 再次调用
```

### 基本语法

```python
def 函数名(参数1, 参数2):
    """文档字符串：说明函数做什么（可选但推荐）"""
    # 函数体
    return 返回值

# 示例
def add(a, b):
    """返回两个数的和"""
    return a + b

result = add(3, 5)    # result = 8
```

## 二、参数类型详解

### 1. 位置参数（最基本）

```python
def greet(name, greeting):
    print(f"{greeting}, {name}!")

greet("张三", "你好")     # 你好, 张三!
greet("你好", "张三")     # 张三, 你好!  ← 顺序搞反了
```

### 2. 关键字参数（指定参数名）

```python
greet(name="张三", greeting="你好")     # ✅
greet(greeting="你好", name="张三")     # ✅ 顺序无所谓了
```

### 3. 默认参数

```python
def greet(name, greeting="你好"):    # greeting 有默认值
    print(f"{greeting}, {name}!")

greet("张三")                # 你好, 张三!（使用默认值）
greet("张三", "Hello")      # Hello, 张三!（覆盖默认值）
```

**⚠️ 经典坑：默认参数不要用可变对象！**

```python
# ❌ 大坑
def add_item(item, items=[]):
    items.append(item)
    return items

print(add_item("a"))    # ['a']
print(add_item("b"))    # ['a', 'b']  ← 共享了同一个列表！

# ✅ 正确做法
def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

### 4. *args（接收任意数量的位置参数）

```python
def my_sum(*args):
    """args 是一个元组"""
    print(type(args))    # <class 'tuple'>
    return sum(args)

print(my_sum(1, 2, 3))        # 6
print(my_sum(1, 2, 3, 4, 5))  # 15
```

### 5. **kwargs（接收任意数量的关键字参数）

```python
def print_info(**kwargs):
    """kwargs 是一个字典"""
    print(type(kwargs))    # <class 'dict'>
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="张三", age=20, city="北京")
# name: 张三
# age: 20
# city: 北京
```

### 6. 参数顺序（面试常问）

```python
def func(a, b, *args, c=10, **kwargs):
    pass

# 顺序：位置参数 → *args → 默认参数 → **kwargs
# 口诀：先位置，再打包，然后默认，最后字典
```

## 三、返回值

### 返回单个值

```python
def square(n):
    return n ** 2

result = square(5)    # 25
```

### 返回多个值（本质是元组）

```python
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder    # 返回元组

q, r = divide(10, 3)    # q=3, r=1
```

### 没有 return 或 return 后面没值

```python
def greet(name):
    print(f"你好, {name}")
    # 没有 return → 返回 None

result = greet("张三")
print(result)    # None
```

## 四、变量作用域

```python
x = 10    # 全局变量

def func():
    x = 20    # 局部变量（和全局的 x 没关系）
    print(x)  # 20

func()
print(x)      # 10（全局的 x 没变）
```

### global 关键字

```python
x = 10

def func():
    global x    # 声明使用全局变量
    x = 20

func()
print(x)    # 20（被修改了）
```

**建议**：尽量避免使用 global，通过参数传入、返回值传出更安全。

## 五、Lambda 匿名函数

```python
# 一行定义简单函数
square = lambda x: x ** 2
print(square(5))    # 25

# 常见用法：作为排序的 key
students = [
    {"name": "张三", "score": 80},
    {"name": "李四", "score": 95},
    {"name": "王五", "score": 70},
]

# 按成绩排序
students.sort(key=lambda s: s["score"])
# 按成绩降序
students.sort(key=lambda s: s["score"], reverse=True)
```

**建议**：lambda 只用于简单的一行逻辑，复杂逻辑用 def 定义。

## 六、常用内置函数

```python
# 数学
print(abs(-5))         # 5
print(max(1, 2, 3))    # 3
print(min(1, 2, 3))    # 1
print(sum([1, 2, 3]))  # 6
print(round(3.456, 2)) # 3.46

# 类型
print(int("42"))       # 42
print(str(42))         # "42"
print(float("3.14"))   # 3.14
print(list("abc"))     # ['a', 'b', 'c']
print(bool(0))         # False

# 序列
print(len([1, 2, 3]))         # 3
print(sorted([3, 1, 2]))      # [1, 2, 3]
print(reversed([1, 2, 3]))    # 迭代器，list() 转换
print(list(zip([1,2], [3,4])))# [(1,3), (2,4)]
print(list(enumerate("abc"))) # [(0,'a'), (1,'b'), (2,'c')]

# 判断
print(isinstance(42, int))    # True
print(all([True, True]))      # True
print(any([False, True]))     # True
```

## 七、文档字符串（Docstring）

```python
def calculate_bmi(weight: float, height: float) -> float:
    """
    计算 BMI 指数。
    
    Args:
        weight: 体重（公斤）
        height: 身高（米）
    
    Returns:
        BMI 指数值
    
    Examples:
        >>> calculate_bmi(70, 1.75)
        22.86
    """
    return round(weight / height ** 2, 2)

# 查看文档
help(calculate_bmi)
print(calculate_bmi.__doc__)
```

**后端开发中**，良好的 docstring 是必须的，它会被自动文档工具（如 FastAPI 的 /docs）直接使用。
