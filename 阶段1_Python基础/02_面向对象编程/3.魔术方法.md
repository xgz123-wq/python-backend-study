# 魔术方法（Magic Methods）

## 一、什么是魔术方法？

魔术方法（也叫双下方法、dunder 方法）是以 `__` 开头和结尾的特殊方法。  
Python 在特定操作时**自动调用**这些方法，让你的类融入 Python 语法。

```python
a + b           # → a.__add__(b)
len(obj)        # → obj.__len__()
print(obj)      # → obj.__str__()
obj[key]        # → obj.__getitem__(key)
with obj:       # → obj.__enter__() 和 obj.__exit__()
```

## 二、最常用的魔术方法

### `__str__` 和 `__repr__`

```python
class Product:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    def __str__(self):
        """print() 和 str() 时调用，面向用户"""
        return f"{self.name} - ¥{self.price:.2f}"

    def __repr__(self):
        """repr() 和调试时显示，面向开发者，要能重现对象"""
        return f"Product(name='{self.name}', price={self.price})"

p = Product("Python书", 59.9)
print(p)        # Python书 - ¥59.90       ← __str__
print(repr(p))  # Product(name='Python书', price=59.9)  ← __repr__

products = [p]
print(products) # 列表中显示用 __repr__
```

**规则**：如果只定义了 `__repr__`，`print()` 也会用它。所以至少定义 `__repr__`。

### `__len__` 和 `__bool__`

```python
class ShoppingCart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def __len__(self):
        """支持 len(cart)"""
        return len(self.items)

    def __bool__(self):
        """支持 if cart:（非空为 True）"""
        return len(self.items) > 0

cart = ShoppingCart()

print(len(cart))     # 0
print(bool(cart))    # False

if not cart:
    print("购物车是空的")

cart.add("苹果")
print(len(cart))     # 1
if cart:
    print("购物车有商品")
```

### `__eq__`、`__lt__` 等比较方法

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def __eq__(self, other):
        """支持 s1 == s2"""
        if not isinstance(other, Student):
            return NotImplemented
        return self.score == other.score

    def __lt__(self, other):
        """支持 s1 < s2，以及 sorted()"""
        if not isinstance(other, Student):
            return NotImplemented
        return self.score < other.score

    def __repr__(self):
        return f"Student({self.name}, {self.score})"

s1 = Student("张三", 90)
s2 = Student("李四", 85)
s3 = Student("王五", 90)

print(s1 == s3)        # True（分数相同）
print(s1 > s2)         # True（自动从 __lt__ 推导）
print(sorted([s1, s2, s3]))  # 按分数排序
```

**`@functools.total_ordering`**：只需要定义 `__eq__` 和一个比较方法，自动补全其他比较：

```python
from functools import total_ordering

@total_ordering
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def __eq__(self, other):
        return self.score == other.score

    def __lt__(self, other):
        return self.score < other.score
    # __le__, __gt__, __ge__ 会自动生成
```

### `__getitem__`、`__setitem__`、`__contains__`

让对象支持下标操作 `[]` 和 `in` 判断：

```python
class Config:
    """支持字典式访问的配置类"""
    def __init__(self):
        self._data = {}

    def __getitem__(self, key):
        """支持 config['key']"""
        return self._data[key]

    def __setitem__(self, key, value):
        """支持 config['key'] = value"""
        self._data[key] = value

    def __contains__(self, key):
        """支持 'key' in config"""
        return key in self._data

    def __repr__(self):
        return f"Config({self._data})"

config = Config()
config["host"] = "localhost"
config["port"] = 3306

print(config["host"])          # localhost
print("host" in config)        # True
print("password" in config)    # False
```

### `__call__`

让对象可以像函数一样被调用：

```python
class Multiplier:
    def __init__(self, factor):
        self.factor = factor

    def __call__(self, x):
        """支持 obj(x) 调用"""
        return x * self.factor

double = Multiplier(2)
triple = Multiplier(3)

print(double(5))    # 10
print(triple(5))    # 15

# 可在需要函数的地方传入对象
numbers = [1, 2, 3, 4, 5]
print(list(map(double, numbers)))   # [2, 4, 6, 8, 10]
```

**实际场景**：机器学习、中间件、装饰器工厂等大量使用 `__call__`。

### `__enter__` 和 `__exit__`（上下文管理器）

让对象支持 `with` 语句，确保资源被正确释放：

```python
class DatabaseConnection:
    def __init__(self, host, port):
        self.host = host
        self.port = port
        self.conn = None

    def __enter__(self):
        """with 语句开始时调用"""
        print(f"连接数据库 {self.host}:{self.port}")
        self.conn = f"Connection({self.host})"
        return self.conn    # return 的值赋给 as 后面的变量

    def __exit__(self, exc_type, exc_val, exc_tb):
        """with 语句结束时调用（无论是否发生异常）"""
        print("关闭数据库连接")
        self.conn = None
        return False    # 返回 False 表示不压制异常

# 使用
with DatabaseConnection("localhost", 3306) as conn:
    print(f"使用连接: {conn}")
    # 即使发生异常，__exit__ 也一定会被调用

# 连接数据库 localhost:3306
# 使用连接: Connection(localhost)
# 关闭数据库连接
```

## 三、常见魔术方法速查表

| 魔术方法 | 触发时机 | 示例 |
|----------|----------|------|
| `__init__` | 创建对象 | `obj = MyClass()` |
| `__str__` | `print(obj)`, `str(obj)` | 用户友好字符串 |
| `__repr__` | `repr(obj)`, 调试器 | 开发者字符串 |
| `__len__` | `len(obj)` | 长度 |
| `__bool__` | `bool(obj)`, `if obj:` | 布尔值 |
| `__eq__` | `obj1 == obj2` | 相等判断 |
| `__lt__` | `obj1 < obj2` | 小于 |
| `__add__` | `obj1 + obj2` | 加法 |
| `__getitem__` | `obj[key]` | 下标访问 |
| `__setitem__` | `obj[key] = val` | 下标赋值 |
| `__contains__` | `x in obj` | 成员判断 |
| `__call__` | `obj(args)` | 像函数一样调用 |
| `__enter__` | `with obj:` 进入 | 上下文管理 |
| `__exit__` | `with obj:` 退出 | 上下文管理 |
| `__iter__` | `for x in obj:` | 迭代 |
| `__next__` | `next(obj)` | 迭代下一个 |

## 四、实际应用：Vector 类

综合运用多个魔术方法，实现一个数学向量类：

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __repr__(self):
        return f"Vector({self.x}, {self.y})"

    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)

    def __sub__(self, other):
        return Vector(self.x - other.x, self.y - other.y)

    def __mul__(self, scalar):
        """向量与标量相乘"""
        return Vector(self.x * scalar, self.y * scalar)

    def __abs__(self):
        """向量模长"""
        return (self.x ** 2 + self.y ** 2) ** 0.5

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

v1 = Vector(1, 2)
v2 = Vector(3, 4)

print(v1 + v2)      # Vector(4, 6)
print(v2 - v1)      # Vector(2, 2)
print(v1 * 3)       # Vector(3, 6)
print(abs(v2))      # 5.0
print(v1 == v1)     # True
```
