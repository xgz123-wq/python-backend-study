# JSON 处理

## 一、为什么需要 JSON？

JSON（JavaScript Object Notation）是后端开发中**最常用的数据交换格式**：

- 前后端通信：API 请求和响应的标准格式
- 配置文件：`package.json`、`settings.json`
- 数据存储：NoSQL 数据库（MongoDB）、日志
- 第三方 API：几乎所有外部服务都使用 JSON

```json
{
    "id": 1,
    "name": "张三",
    "email": "zs@qq.com",
    "skills": ["Python", "FastAPI"],
    "is_active": true,
    "address": null
}
```

**JSON 与 Python 数据类型的对应**：

| JSON | Python |
|------|--------|
| `{}` 对象 | `dict` |
| `[]` 数组 | `list` |
| `"string"` | `str` |
| `123` / `3.14` | `int` / `float` |
| `true` / `false` | `True` / `False` |
| `null` | `None` |

## 二、json 模块基础

### json.dumps() — Python 对象 → JSON 字符串

```python
import json

user = {
    "name": "张三",
    "age": 20,
    "skills": ["Python", "FastAPI"],
    "is_active": True,
    "address": None
}

# 转成 JSON 字符串
json_str = json.dumps(user)
print(json_str)
# {"name": "张三", "age": 20, "skills": ["Python", "FastAPI"], "is_active": true, "address": null}

# 注意：True → true，None → null，中文变成了 \uXXXX
```

### 常用参数

```python
# ensure_ascii=False：保留中文
json_str = json.dumps(user, ensure_ascii=False)
print(json_str)
# {"name": "张三", "age": 20, ...}

# indent：格式化输出（缩进）
json_str = json.dumps(user, ensure_ascii=False, indent=2)
print(json_str)
# {
#   "name": "张三",
#   "age": 20,
#   ...
# }

# sort_keys：按键排序
json_str = json.dumps(user, ensure_ascii=False, indent=2, sort_keys=True)
```

### json.loads() — JSON 字符串 → Python 对象

```python
import json

json_str = '{"name": "张三", "age": 20, "is_active": true, "address": null}'
data = json.loads(json_str)

print(data)            # {'name': '张三', 'age': 20, 'is_active': True, 'address': None}
print(type(data))      # <class 'dict'>
print(data["name"])    # 张三

# true → True，null → None，自动转换
```

## 三、文件读写 JSON

### json.dump() — 写入文件

```python
import json

users = [
    {"id": 1, "name": "张三", "email": "zs@qq.com"},
    {"id": 2, "name": "李四", "email": "ls@qq.com"},
]

with open("users.json", "w", encoding="utf-8") as f:
    json.dump(users, f, ensure_ascii=False, indent=2)
```

### json.load() — 从文件读取

```python
import json

with open("users.json", "r", encoding="utf-8") as f:
    users = json.load(f)

for user in users:
    print(f"{user['id']}: {user['name']} <{user['email']}>")
```

### dumps/loads vs dump/load 对比

| 函数 | 输入/输出 | 用途 |
|------|-----------|------|
| `json.dumps(obj)` | Python → 字符串 | API 响应、网络传输 |
| `json.loads(str)` | 字符串 → Python | 解析 API 响应 |
| `json.dump(obj, file)` | Python → 文件 | 保存到文件 |
| `json.load(file)` | 文件 → Python | 从文件读取 |

> 📝 **记忆**：带 `s` 的是 **s**tring（字符串），不带 `s` 的是文件。

## 四、处理特殊类型

JSON 不支持所有 Python 类型，需要自定义序列化：

```python
import json
from datetime import datetime, date
from pathlib import Path

# ❌ 直接序列化会报错
data = {"created_at": datetime.now()}
# json.dumps(data)   # TypeError: Object of type datetime is not JSON serializable

# ✅ 方式一：自定义编码器
class CustomEncoder(json.JSONEncoder):
    def default(self, obj):
        if isinstance(obj, (datetime, date)):
            return obj.isoformat()
        if isinstance(obj, Path):
            return str(obj)
        if isinstance(obj, set):
            return list(obj)
        return super().default(obj)

data = {
    "created_at": datetime.now(),
    "config_path": Path("/etc/app/config.json"),
    "tags": {"python", "backend"},
}

json_str = json.dumps(data, cls=CustomEncoder, ensure_ascii=False, indent=2)
print(json_str)

# ✅ 方式二：用 default 参数（简单场景）
json_str = json.dumps(data, default=str, indent=2)   # 所有不认识的类型都转成 str
```

## 五、后端实际场景

### API 响应构造

```python
import json

def make_response(data, code=200, message="success"):
    """构造标准 API 响应"""
    response = {
        "code": code,
        "message": message,
        "data": data
    }
    return json.dumps(response, ensure_ascii=False)

# 成功响应
print(make_response({"id": 1, "name": "张三"}))
# {"code": 200, "message": "success", "data": {"id": 1, "name": "张三"}}

# 错误响应
print(make_response(None, code=404, message="用户不存在"))
# {"code": 404, "message": "用户不存在", "data": null}
```

### 配置文件管理

```python
import json
from pathlib import Path

class Config:
    """简单的 JSON 配置管理器"""
    def __init__(self, filepath: str):
        self.path = Path(filepath)
        self.data = {}
        self._load()

    def _load(self):
        if self.path.exists():
            self.data = json.loads(self.path.read_text(encoding="utf-8"))

    def save(self):
        self.path.write_text(
            json.dumps(self.data, ensure_ascii=False, indent=2),
            encoding="utf-8"
        )

    def get(self, key, default=None):
        return self.data.get(key, default)

    def set(self, key, value):
        self.data[key] = value
        self.save()

# 使用
# config = Config("app_config.json")
# config.set("debug", True)
# config.set("db_host", "localhost")
# print(config.get("debug"))   # True
```

### 解析第三方 API 响应

```python
import json

# 模拟第三方 API 返回的 JSON 字符串
api_response = '''
{
    "status": "success",
    "data": {
        "users": [
            {"id": 1, "name": "张三", "role": "admin"},
            {"id": 2, "name": "李四", "role": "user"}
        ],
        "total": 2,
        "page": 1
    }
}
'''

result = json.loads(api_response)

# 安全取值：用 get() 避免 KeyError
users = result.get("data", {}).get("users", [])
for user in users:
    print(f"  {user['id']}: {user['name']} ({user['role']})")
```

## 六、常见坑

### 坑1：中文变成 \uXXXX

```python
# ❌ 默认 ensure_ascii=True
json.dumps({"name": "张三"})
# '{"name": "\\u5f20\\u4e09"}'

# ✅ 加 ensure_ascii=False
json.dumps({"name": "张三"}, ensure_ascii=False)
# '{"name": "张三"}'
```

### 坑2：JSON 键必须是字符串

```python
# ❌ Python dict 的键可以是 int，但 JSON 不行
data = {1: "one", 2: "two"}
json_str = json.dumps(data)
result = json.loads(json_str)
print(result)   # {"1": "one", "2": "two"}  ← 键变成了字符串！
```

### 坑3：JSON 不支持单引号

```python
# ❌ 单引号不是合法 JSON
json.loads("{'name': 'zhangsan'}")   # JSONDecodeError

# ✅ JSON 必须用双引号
json.loads('{"name": "zhangsan"}')   # OK
```

### 坑4：json.loads 接收字符串，json.load 接收文件对象

```python
# ❌ 混淆了 loads 和 load
with open("data.json") as f:
    data = json.loads(f)    # TypeError: 应该传字符串，不是文件对象

# ✅ 从文件读用 load（不带 s）
with open("data.json") as f:
    data = json.load(f)

# ✅ 从字符串读用 loads（带 s）
data = json.loads('{"key": "value"}')
```

### 坑5：JSON 不支持注释

```python
# ❌ 这不是合法 JSON
bad_json = '''
{
    // 这是注释
    "name": "张三"
}
'''
json.loads(bad_json)   # JSONDecodeError

# JSON 标准不支持注释，如需注释配置可用 YAML 或 TOML
```
