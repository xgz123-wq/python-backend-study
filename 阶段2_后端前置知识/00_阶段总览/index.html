2学习主线

阶段 2 的内容不是并列碎片，而是一条后端请求链路。

① 网络基础

客户端请求先经过 DNS、TCP/IP、端口定位，找到目标服务。

② HTTP 协议

请求进入服务后，以 HTTP 方法、Header、Body、状态码表达意图与结果。

③ RESTful API

把 HTTP 能力组织成稳定、可维护、可演进的接口设计规范。

④ 认证授权

在接口访问前后判断用户身份、权限边界与安全凭证。
