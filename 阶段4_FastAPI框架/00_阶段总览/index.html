4环境准备（3 步启动 demo）

Python 3.10+ 即可，无需 MySQL——demo 默认用 SQLite

步骤命令作用
1. 装依赖cd 01_FastAPI核心/demo && pip install -r requirements.txtfastapi / uvicorn / sqlalchemy / aiosqlite 等
2. 启服务uvicorn app.main:app --reload启动 FastAPI（带热重载）
3. 打开文档http://localhost:8000/docs浏览器看自动生成的 Swagger UI
⚠️ 启动注意

必须在 demo/ 目录下运行 uvicorn，否则 ModuleNotFoundError: No module named 'app'。换 MySQL 只需改 database.py 里的 DATABASE_URL 为 mysql+aiomysql://...。
