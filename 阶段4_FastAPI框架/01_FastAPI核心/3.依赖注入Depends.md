# 依赖注入 Depends

> 把数据库连接、登录校验抽成函数，每个路由自动复用，不再复制粘贴。

**一句话定义**：`Depends()` = 把公共逻辑（如获取数据库连接、校验 JWT）抽成函数，路由自动调用并拿到结果，不用每个路由重复写。

## 为什么需要

没有依赖注入时，每个路由都要手动写：
```python
# ❌ 每个路由都重复
@app.get("/items/")
async def list_items():
    db = SessionLocal()
    try:
        ...
    finally:
        db.close()
```

有了 `Depends()`：
```python
# ✅ 抽出来，复用
async def get_db():
    db = AsyncSession(engine)
    try:
        yield db          # yield 让 FastAPI 在请求结束后自动 close
    finally:
        await db.close()

@app.get("/items/")
async def list_items(db: AsyncSession = Depends(get_db)):
    ...  # db 已经准备好了

@app.post("/items/")
async def create_item(item: ItemCreate, db: AsyncSession = Depends(get_db)):
    ...  # 同一个 db 注入机制，不用重写
```

## 常见用途

```python
# 1. 数据库 session
async def get_db(): ...

# 2. 获取当前登录用户（JWT 校验）
async def get_current_user(token: str = Depends(oauth2_scheme), db=Depends(get_db)):
    payload = decode_jwt(token)
    return await db.get(User, payload["sub"])

# 3. 路由中使用
@app.get("/me")
async def get_me(user: User = Depends(get_current_user)):
    return user
```

## 1 个常见坑

**`yield` vs `return`**：依赖函数里用 `yield` 才能在请求结束后执行清理（如关闭 db）；用 `return` 就没有清理钩子。数据库 session 必须用 `yield`。
