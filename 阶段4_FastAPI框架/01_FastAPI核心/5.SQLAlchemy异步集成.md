# SQLAlchemy 异步集成

> 给 ORM 换上异步引擎，FastAPI 才能真正并发访问数据库。

**一句话定义**：用 `AsyncSession` + 异步驱动（`aiosqlite`/`asyncpg`）让 SQLAlchemy 在 FastAPI 中非阻塞地操作数据库。

## 最小代码片段

```python
# database.py
from sqlalchemy.ext.asyncio import create_async_engine, AsyncSession, async_sessionmaker
from sqlalchemy.orm import DeclarativeBase

# SQLite 异步（开发用，无需安装 MySQL）
DATABASE_URL = "sqlite+aiosqlite:///./app.db"
# MySQL 异步（生产用）
# DATABASE_URL = "mysql+aiomysql://root:password@localhost:3306/mydb"

engine = create_async_engine(DATABASE_URL, echo=True)
AsyncSessionLocal = async_sessionmaker(engine, expire_on_commit=False)

class Base(DeclarativeBase):
    pass

# models.py
from sqlalchemy.orm import Mapped, mapped_column
from database import Base

class Item(Base):
    __tablename__ = "items"
    id: Mapped[int] = mapped_column(primary_key=True)
    name: Mapped[str] = mapped_column(nullable=False)
    price: Mapped[float]

# deps.py - DB session 依赖
async def get_db():
    async with AsyncSessionLocal() as session:
        yield session   # 请求结束自动 close

# routers/items.py - 路由中使用
from sqlalchemy import select

@app.get("/items/")
async def list_items(db: AsyncSession = Depends(get_db)):
    result = await db.execute(select(Item))
    return result.scalars().all()

@app.post("/items/")
async def create_item(item: ItemCreate, db: AsyncSession = Depends(get_db)):
    db_item = Item(**item.model_dump())
    db.add(db_item)
    await db.commit()
    await db.refresh(db_item)
    return db_item
```

## 建表（启动时自动建）

```python
# main.py 启动事件
@app.on_event("startup")
async def startup():
    async with engine.begin() as conn:
        await conn.run_sync(Base.metadata.create_all)
```

## 1 个常见坑

**`expire_on_commit=False`**：commit 之后 ORM 对象默认过期（访问属性会触发新查询），异步环境下这会报错。`async_sessionmaker` 传 `expire_on_commit=False` 可避免。

> 对应 demo：`demo/app/database.py` + `demo/app/models/item.py`
