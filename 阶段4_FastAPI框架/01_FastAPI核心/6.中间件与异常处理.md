# 中间件与异常处理

> 给每个请求装一层拦截器，统一打日志、统一返错，不再泄露 500 堆栈。

**一句话定义**：中间件 = 每个请求进来/出去都会经过的拦截层（记日志、加响应头）；异常处理 = 把程序错误转成统一 JSON 格式返回，而不是让客户端看到 500 堆栈。

## 中间件

```python
import time
from fastapi import Request

# 自定义中间件：记录每个请求耗时
@app.middleware("http")
async def add_process_time(request: Request, call_next):
    start = time.time()
    response = await call_next(request)   # 调用下一个处理层
    process_time = time.time() - start
    response.headers["X-Process-Time"] = str(process_time)
    return response

# CORS 中间件（前后端分离必须加）
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],  # 生产改为实际前端域名
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 统一异常处理

```python
from fastapi import HTTPException
from fastapi.responses import JSONResponse

# 自定义业务异常
class ItemNotFound(Exception):
    def __init__(self, item_id: int):
        self.item_id = item_id

# 注册异常处理器
@app.exception_handler(ItemNotFound)
async def item_not_found_handler(request, exc):
    return JSONResponse(
        status_code=404,
        content={"code": 404, "message": f"Item {exc.item_id} not found"}
    )

# 路由里直接 raise，不用写 if/return
@app.get("/items/{item_id}")
async def get_item(item_id: int, db: AsyncSession = Depends(get_db)):
    item = await db.get(Item, item_id)
    if not item:
        raise ItemNotFound(item_id)   # 自动走 handler，返回 404 JSON
    return item
```

## 1 个常见坑

**CORS 必须在其他中间件之前加**：`add_middleware` 是后进先出的，CORS 必须最后 `add`（即代码写在最前面），否则预检请求（OPTIONS）会被其他中间件拦截，前端报 CORS 错误。
