第一节 · Linux 基础命令

服务器命令行是后端的工作台，先把目录、文件、管道、重定向这些日常操作练熟。
🎯 学习目标▼
  • 理解 Linux 发行版与目录树。
  • 掌握文件、目录、搜索、查看命令。
  • 理解管道和重定向。
📖 核心概念▼
/ 根目录
├── etc   配置
├── var   日志/数据
├── home  用户目录
└── usr   用户安装程序

Linux 用一棵以 `/` 为根的目录树组织文件；后端最常接触 `/etc/nginx/`、`/var/log/`、`/home/`、`/opt/`。

💻 代码示例▼

来源：demo/1_basic_commands.py

result = subprocess.run(
    "ls -la",
    shell=True,
    capture_output=True,
    text=True,
    encoding="utf-8",
)
⚠️ 常见误区▼
不要执行 `rm -rf /` 或 `rm -rf *` 这类不可恢复删除命令。
✏️ 自测题▼
Linux 文件系统根目录是？ A. C:\ B. / C. home
实时查看日志常用？ A. tail -f B. rm -rf C. chmod 777
管道符是？ A. > B. & C. |

第二节 · 权限与用户管理

多用户系统里，rwx 与用户组决定谁能动什么，是部署安全的第一道闸。
🎯 学习目标▼
  • 理解 rwx 与 755/644。
  • 掌握用户、用户组、sudo。
  • 知道 SSH 密钥权限要求。
📖 核心概念▼
权限数字含义
r4读取
w2写入
x1执行/进入目录

755 = owner 读写执行，group/other 读执行；644 = owner 读写，其他只读。

💻 代码示例▼

来源：demo/2_permissions.py

mode = os.stat(path).st_mode
permissions = oct(mode)[-3:]
os.chmod(path, 0o644)
⚠️ 常见误区▼
误区纠正
生产直接 chmod 777严重扩大攻击面，应按最小权限。
SSH 私钥可公开读私钥通常应为 600。
✏️ 自测题▼
rwx 数字分别是？ A. 4/2/1 B. 1/2/3 C. 7/6/5
SSH 私钥常用权限？ A. 777 B. 600 C. 000
临时提权命令？ A. tail B. grep C. sudo

第三节 · 进程管理

你的服务在服务器上就是一个进程，能看、能起、能停、能查端口才算掌控。
🎯 学习目标▼
  • 理解 PID、前台/后台进程。
  • 掌握 ps/top/kill/nohup。
  • 了解 systemd 服务管理。
📖 核心概念▼
创建 → 就绪 → 运行 → 阻塞 → 终止
PID 标识进程，PPID 标识父进程
💻 代码示例▼

来源：demo/3_process_management.py

proc = subprocess.Popen([sys.executable, "worker.py"])
print(proc.pid)
proc.terminate()
proc.wait(timeout=5)
⚠️ 常见误区▼

终止进程先用 SIGTERM，确认无法退出再用 SIGKILL；`kill -9` 是最后手段。

✏️ 自测题▼
PID 是？ A. 端口号 B. 进程 ID C. 用户名
断开终端仍运行常用？ A. nohup B. cat C. pwd
现代服务管理器？ A. pip B. npm C. systemd

第四节 · 网络与文本处理

curl、netstat、grep、awk——线上排错就是靠这套组合从日志里找答案。
🎯 学习目标▼
  • 会用 curl/wget/ss/ip 做网络排查。
  • 理解端口、防火墙、监听。
  • 掌握 grep/sed/awk 思路。
📖 核心概念▼
工具作用
curlHTTP/API 调试
ss查看监听端口
grep搜索文本
awk按列统计分析
💻 代码示例▼

来源：demo/4_network_text.py

with socket.create_connection((host, port), timeout=2):
    print("端口可达")

matches = re.findall(r"ERROR|WARN", log_text)
⚠️ 常见误区▼

不要把数据库端口（如 3306、6379）直接暴露到公网；优先只监听内网或本机。

✏️ 自测题▼
测试 API 常用？ A. curl B. chmod C. useradd
查看监听端口现代工具？ A. tar B. ss C. zip
统计日志 IP 访问量常用组合？ A. echo B. pwd C. awk/sort/uniq
