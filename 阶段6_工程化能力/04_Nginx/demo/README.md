# Nginx - Demo 学习指南

## 学习顺序

按编号顺序依次学习，每个 Demo 对应一个理论知识点。

---

### Demo 1: Nginx 基础与反向代理（理解请求转发原理）
**文件**: `1_nginx_basics.py`

用纯 Python 实现简单的反向代理服务器，演示正向代理和反向代理的区别，模拟请求转发过程，展示负载均衡轮询逻辑。帮助理解 Nginx 反向代理的底层工作原理。

**运行方式**: `python 1_nginx_basics.py`
**前置依赖**: 无额外依赖（仅使用标准库）

---

### Demo 2: 静态文件与 HTTPS（理解文件托管和加密通信）
**文件**: `2_static_https.py`

用 Python 搭建简单的静态文件服务器，展示 gzip 压缩效果，模拟 HTTPS/TLS 握手过程，演示证书验证原理。帮助理解 Nginx 处理静态文件和 HTTPS 的底层机制。

**运行方式**: `python 2_static_https.py`
**前置依赖**: 无额外依赖（仅使用标准库）

---

### Demo 3: 负载均衡器（理解请求分发算法）
**文件**: `3_load_balancer.py`

模拟完整的负载均衡器：启动多个后端实例，实现轮询、权重、最少连接三种算法，模拟健康检查机制，展示请求分发效果。帮助深入理解 Nginx upstream 的工作原理。

**运行方式**: `python 3_load_balancer.py`
**前置依赖**: 无额外依赖（仅使用标准库）

---

### 配置文件: nginx.conf 完整示例
**文件**: `nginx.conf`

一份带有详细中文注释的完整 Nginx 配置文件，涵盖反向代理 FastAPI、静态文件托管、HTTPS 配置、负载均衡 upstream、gzip 压缩等核心功能。可直接作为生产环境的配置模板参考。

**使用方式**: 阅读注释理解各配置项含义，按需复制到 `/etc/nginx/conf.d/` 中使用

## 核心要点

1. **反向代理 = 服务端的代理**：客户端不知道后端服务器的存在
2. **正向代理 = 客户端的代理**：代理代表客户端访问目标服务器
3. **静态文件交给 Nginx**：性能比 Python 处理高数十倍
4. **HTTPS = HTTP + TLS**：非对称加密传递密钥，对称加密传输数据
5. **负载均衡算法选择**：后端相同用轮询，性能不同用权重，需 Session 粘性用 IP Hash
6. **改完配置必做**：`nginx -t` 测试语法 + `nginx -s reload` 重载

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| `nginx -t` 报语法错误 | 检查分号、大括号是否遗漏 |
| 502 Bad Gateway | 确认后端服务已启动且端口正确 |
| 静态文件 403 | 检查文件目录权限和 Nginx 运行用户 |
| WebSocket 60 秒断开 | 增大 `proxy_read_timeout` |
| 上传文件 413 | 增大 `client_max_body_size` |
